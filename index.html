📋 ЧТО ДОБАВИТЬ ДЛЯ ФИНАЛЬНОЙ ВЕРСИИ САЙТА

  • Фотографии блюд (hero + карточки меню): макро-съёмка на нейтральном фоне, дневной свет или студийный свет. Форматы: JPG/WebP, минимум 1200×1200 px для карточек, 1800×1200 для hero.
  • Видео-ролик для hero: короткое (10–20 сек) ambient-видео — кофе льётся, десерт разрезают, атмосфера зала. Можно зациклить без звука.
  • Фото интерьера кофейни: 3–5 широких кадров зала, витрины, деталей декора — для раздела «О нас».
  • Логотип в SVG: загрузить векторный файл логотипа для шапки и футера вместо текста.
  • Реальный адрес и карта: заменить заглушку на виджет Яндекс.Карт или Google Maps с меткой.
  • Фото команды / бариста: живые портреты — добавляет доверия и тепла бренду.
  • Отзывы (раздел): скриншоты или текст реальных отзывов с Google / Яндекс / Instagram-хайлайтс «Отзывы».
  • Сертификат «Хорошее место» от Яндекса: использовать как бейдж доверия на главной странице.
  • Декор для акций/сезонного меню: плашки, стикеры, анимированные бейджи («Хит сезона», «Новинка», «Лимитировано»).
